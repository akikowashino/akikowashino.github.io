# MIDI2MIDI

**MIDI2MIDI is a system designed to align (automatically synchronize) two different performance MIDI data tracks of the same piece, enabling users to precisely detect and map specific note information defined in one MIDI to the other with a one-to-one correspondence.**

When analyzing multiple performance MIDI data of the same piece, manually extracting and comparing the exact onset timings of specific musical events—such as the downbeat of each measure or the accompaniment pattern in a triple meter—has traditionally been a labor-intensive and repetitive task.

With MIDI2MIDI, users only need to perform the annotation process (labeling or note designation) on a single reference MIDI track. The system then automatically propagates these annotations to all other performance MIDI data using alignment technology. This completely eliminates tedious, repetitive manual labor, allowing for a highly efficient and precise comparative analysis of performance behaviors.
